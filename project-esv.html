Visualizing Cancer Genomics

Interactive Visualization Framework

A framework that helps oncologists build interactive graphs and tables on top of filtered genomic datasets. This project utilizes ElasticSearch for extremely fast queries on large denormalized datasets to support realtime filtering of coverage vs allele ratio heatmaps, copy number alteration plots, and other views. This framework is applicable to multiple data types, easily altered via a config file.
